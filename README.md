This is my neovim configuration minimal, modern and functional.

**Installation**

```bash
mkdir -p ~/.config/nvim 
git clone https://github.com/DynamiteMonki/nvim ~/.config/nvim/ && nvim 
```

**uninstalltion**

```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
```
